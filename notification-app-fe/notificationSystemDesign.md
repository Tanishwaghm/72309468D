# Notification Priority System Design

## Objective

The objective is to display the Top 10 priority notifications efficiently.

---

# Frontend

React with TypeScript is used for the frontend implementation.

Styling is done using Vanilla CSS.

---

# Priority Logic

Notifications are ranked using:

- Notification type
- Important keywords
- Priority score

---

# Efficient Top 10 Maintenance

A Min Heap approach is used to efficiently maintain Top 10 notifications.

Time Complexity:

O(N log 10)

Approximately O(N)

---

# Conclusion

The system efficiently prioritizes notifications
using optimized data structures.
